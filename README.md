# ai project 1

Project for NNU Artificial Intelligence course

## Features

- Knapsack Problem Solver using Simulated Annealing
- Checkers Game with Alpha-Beta Algorithm

## How to use?

```shell
# install dependencies
$ npm install

# run development server
$ npm run dev
```
